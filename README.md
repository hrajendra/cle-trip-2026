# CLE Trip 2026 — Harsha & Preetham

A Progressive Web App for a brothers' weekend in Cleveland, Ohio. May 8–10, 2026.

**Live app:** https://hrajendra.github.io/cle-trip-2026

## What's inside

- **Home** — countdown timer, nav, quick reference strip
- **Prep** — booking checklist (persistent state), packing list, arrival instructions
- **Day 1 (Fri May 8)** — Soho Chicken + Whiskey, The Whiskey Bar
- **Day 2 (Sat May 9)** — Brandywine Falls → Teddy Baldassarre → Masthead Brewing → Griffin Gin Bar
- **Day 3 (Sun May 10)** — Ledges Trailhead → The South Side → CLE airport
- **Food & Drink Guide** — every must-order, scotch and gin strategy
- **Quick Reference** — emergency numbers, drive times, venue contacts

## PWA features

- Offline-first service worker — all pages cached on first visit
- Add to Home Screen on iOS (Safari → Share → Add to Home Screen)
- Dark theme throughout

## Built with

Claude Sonnet 4 + GitHub Push skill. Based on the pattern from [sg-trip-2026](https://github.com/hrajendra/sg-trip-2026).
